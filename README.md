# BigGirl

#### 项目介绍：
[BigGirl](https://github.com/1900Star/BigGirl)  一个集图片浏览、开发技术干货和音乐播放的综合资讯客户端。<br>


## 数据来源
   * [干货集中营](http://gank.io/)
   * [Unsplash](http://www.unsplash.com/)
